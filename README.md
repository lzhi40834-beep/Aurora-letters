# 极光信笺 — Aurora Letters 🌌

北欧诗意交友网站。极光、森林、峡湾、信笺——在极光下遇见有趣的人。

## 设计风格

- **主题**：北欧极光 + 挪威森林 + 小木屋 + 韩系高级感
- **色调**：松木深绿 + 峡湾雾灰 + 极光绿/紫/蓝 + 壁炉暖金
- **字体**：Cormorant Garamond（英文衬线）+ Noto Serif SC（中文衬线）+ Noto Serif KR（韩文衬线）+ Noto Sans SC（正文）
- **背景**：CSS多层森林渐变 + Canvas极光动画 + 玻璃拟态卡片
- **音乐**：Web Audio API 实时生成环境音（无需外部音频文件）

## 功能

| 页面 | 说明 |
|---|---|
| Landing | 全屏极光动画 + 三语诗意引言（中/英/韩）+ 玻璃卡片 |
| 注册 | 3步向导：年龄 → 选择头像 → 名字 |
| 主页 | 个人资料 + MBTI结果展示 + 快捷导航 |
| MBTI测试 | 20题人格测试，自动计分，诗意结果描述 |
| 发现 | 8位预设北欧人物，可浏览资料并发信 |
| 极光信笺 | 收件箱/已寄出双Tab，信件状态流转，自动回复 |

## 技术

- 纯前端，无框架，无后端
- 数据全存 localStorage
- Canvas 极光引擎（6条缎带 + 150星 + 雾粒子）
- Web Audio API 环境音生成（4层drone + 峡湾风声）
- 玻璃拟态（Glassmorphism）+ CSS多层渐变背景
- URL hash 路由，支持浏览器前进后退
